# TPO Management - User Guide with Use Cases

## Overview

This comprehensive user guide provides detailed use cases for each module function in the TPO Management system. Each use case includes step-by-step instructions, real-world scenarios, and best practices.

## Use Case Format

Each use case document follows this structure:

1. **Overview** - Purpose and scope
2. **Use Cases** - Multiple scenarios numbered (e.g., Use Case X.1, X.2)
3. **Steps** - Detailed step-by-step instructions
4. **Expected Result** - What should happen after completion
5. **Best Practices** - Tips and recommendations
6. **Related Use Cases** - Links to related functionality
7. **Troubleshooting** - Common issues and solutions (when applicable)

## Getting Started

If you're new to the system:

1. **Start Here**: [Project Setup](01_project_management/project_setup.md)
   - Learn to create and configure projects

2. **Then Follow Your Project Phase**:
   - **New Project**: Start with [BOQ](02_pre_execution_planning/boq.md) and [Project Schedule](02_pre_execution_planning/project_schedule.md)
   - **During Construction**: Use [Shop Drawings](03_execution_monitoring/shop_drawing.md), [RFIs](03_execution_monitoring/rfi.md), [ITRs](03_execution_monitoring/itr.md)
   - **Near Completion**: Work on [As-Built Drawings](04_financial_closure/as_built_drawing.md) and [Project Handover](04_financial_closure/project_handover.md)

3. **Use Search**: If you know what you need, use your browser's search (Ctrl+F) to find specific topics
