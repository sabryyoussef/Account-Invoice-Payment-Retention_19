# TPO Construction Management - Odoo Modules

This repository contains Odoo 19 Community Edition modules for Technical Project Office (TPO) Construction Management.

## Modules

### tpo_management
Comprehensive document management and work control for Technical Project Office (TPO) operations in construction projects.

**Features:**
- Pre-Execution & Planning Phase Documents (BOQ, Project Schedule, Material Submittal, Technical Offers)
- Execution & Monitoring Phase Documents (Shop Drawings, RFIs, ITRs, Change Orders, Correspondence Log)
- Financial & Closure Phase Documents (Payment Applications, As-Built Drawings, Project Handover)
- TPO Site Visit Management
- Document Control System (DCC)

**Version:** 19.0.1.0.0

### tpo_reports
Detailed reporting capabilities for Technical Project Office (TPO) operations.

**Features:**
- BOQ Reports
- Shop Drawing Status Reports
- RFI Tracking Reports
- ITR Status Reports
- Change Order Reports
- Payment Application Reports
- Document Status Dashboard
- Submission Log Reports
- Project Progress Reports

**Version:** 19.0.1.0.0

## Requirements

- Odoo 19 Community Edition
- PostgreSQL 15+
- Python 3.10+

## Installation

1. Clone this repository
2. Copy the modules to your Odoo addons path
3. Update the apps list in Odoo
4. Install the modules from the Apps menu
